# Rust File Manager

Metin tabanlı basit bir dosya yöneticisidir, Rust dilinin standart kütüphanesi üzerinde bulunan fonksiyonlar yardımıyla oluşturulmuştur.
Günlük kullanım projesi değildir eğitim amaçlı kullanılmıştır.

Temel Özellikler
- Dosya silme
- Dosya oluşturma
- Dosya okuma
- Klasör silme
- Klasör oluşturma
